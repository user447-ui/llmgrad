A tiny efficient deep learning framework written in C++ with Python bindings.
It implements Tensor Autograd, matrix multiplication, and a PyTorch-like API (`nn.Module`, `optim.SGD`).

## Features
*   **Core:** Written in C++ for performance.
*   **Autograd:** Reverse-mode automatic differentiation (Backprop).
*   **API:** Looks exactly like PyTorch.
*   **Context:** Supports embedding concatenation for NLP tasks.

## Installation

pip install cllm
